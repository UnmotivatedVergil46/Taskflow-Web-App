===============================================
TROUBLESHOOTING
===============================================

Problem: ModuleNotFoundError
   Solution: pip install -r requirements.txt

Problem: Port 5000 already in use
   Solution: PORT=3000 python main.py

Problem: MySQL connection error
   Solution: App uses in-memory storage automatically
   (No MySQL required, optional for production)

Problem: Frontend can't connect
   Solution: CORS is enabled by default
   Check that your frontend API URL is correct
